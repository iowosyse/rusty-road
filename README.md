# Rusty Road

Aquí estaré aprendiendo el lenguaje de programación [Rust](https://github.com/rust-lang/rust), recorriendo nuevamente el camino que seguí cuando aprendí a programar por primera vez durante mi primer semestre de la universidad. 

Existen 2 directorios principales:
- [lecciones](lecciones/)   
  Aquí es donde irán los ejercicios de ejemplo y donde tendré contacto con la sintaxis de Rust por primera vez.
- [problemas](problemas/)   
  Dentro de esta carpeta hay un archivo llamado [problemas](problemas/problemas.md), en el cual está almacenada una antología de problemas de programación de nivel básico, que exploran todo tipo de situaciones, tales como:
  - Problemas de entrada y salida desde terminal
  - Variables y tipos de datos
  - Arreglos y matrices
  - Conceptos de POO
  - Estructuras de datos
